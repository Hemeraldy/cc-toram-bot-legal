Política de Privacidad

Bot: C.C-Toram-Bot

Última actualización: 2026

Datos recopilados

C.C-Toram-Bot puede recopilar ciertos datos necesarios para su funcionamiento:

  • ID de usuario
  • ID de servidor
  • ID de canal
  • Configuraciones del servidor

Uso de los datos

Los datos recopilados se utilizan únicamente para el funcionamiento del bot.

Compartición de datos

C.C-Toram-Bot no vende ni comparte información con terceros.

Seguridad

Se toman medidas razonables para proteger la información almacenada.

Derechos del usuario

Los administradores de servidores pueden solicitar la eliminación de datos asociados a su servidor.

Contacto

Si tienes preguntas sobre esta política puedes contactar al desarrollador mediante Discord.
